# Pinyin Interpreter

A python script that converts ascii pinyin text to unicode pinyin!!!

Example Usage:
<pre>
$ pinyin han4 yu3 pin1 yin1
hàn yǔ pīn yīn

$ pinyin ha4nyu3 pi1nyi1n
hànyǔ pīnyīn

$ echo han4yu3 pin1yin1 | pinyin
hànyǔ pīnyīn
</pre>
